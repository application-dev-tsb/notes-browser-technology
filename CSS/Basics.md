# Basics: Cascading Style Sheets

#### Import


#### Selector
```css
//select all
* {
  font-family: "Arial Black', sans-serif;
}

//select by id
#myId {

}

//div>span>a
div span a {

}

//div OR span OR a
div, span, a {

}
```
