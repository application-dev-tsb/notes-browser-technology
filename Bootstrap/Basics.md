# Basics: Bootstrap
- HTML5
- Mobile first
- Responsive CSS for Mobile and Desktop

#### 12-Column Grid System
- elements consume **x** out of **12** columns for each **row** based off the screen size
```html
<div class="row">
  <div class="col-lg-8 col-md-6 col-sm-1">
  </div>
</div>
```
