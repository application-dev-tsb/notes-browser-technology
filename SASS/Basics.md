# Basics: Syntactically Awesome Style Sheets

#### Comments
- single line
```sass
// singleline comment
```
- multiline
```sass
/*
Im 
a 
multiline comment
*/
```
- loud comments
```sass
/*! Loud comments are quaranteed to show up in the generated css */
```
